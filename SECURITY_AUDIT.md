## 高危漏洞修复详情

### 漏洞 1: RUSTSEC-2026-0195

| 项目 | 内容 |
|------|------|
| Crate | quick-xml |
| 版本 | v0.39.4 |
| 标题 | Unbounded namespace-declaration allocation in `NsReader` enables memory-exhaustion denial of service |
| 严重程度 | 7.5 (High) |
| 修复方案 | 升级至 >=0.41.0 |

**影响分析：**
- 攻击者可通过构造恶意 XML 输入导致内存耗尽
- 本应用使用 quick-xml 作为间接依赖（plist → quick-xml）
- 攻击向量有限，因为应用不直接解析用户提供的 XML

**修复过程：**
```bash
cargo update  # 自动升级 quick-xml 至 v0.41.0
```

### 漏洞 2: RUSTSEC-2026-0194

| 项目 | 内容 |
|------|------|
| Crate | quick-xml |
| 版本 | v0.39.4 |
| 标题 | Quadratic run time when checking a start tag for duplicate attribute names |
| 严重程度 | 7.5 (High) |
| 修复方案 | 升级至 >=0.41.0 |

**影响分析：**
- 攻击者可通过构造包含大量重复属性名的 XML 导致拒绝服务
- 同样通过 plist 间接依赖引入

**修复过程：**
与漏洞1同时修复，通过 `cargo update` 升级至 v0.41.0
